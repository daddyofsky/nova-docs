# Nova 프레임워크 매뉴얼
Nova Framework Manual

## 소개

Nova 프레임워크는 라라벨을 참고하여 개발된 PHP 프레임워크로,
개발의 편의성을 지원하면서도 간단하고 빠른 성능을 제공하는 것을 목표로 합니다.
